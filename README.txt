📘 EuroMillions Master Wizard v1.0 – Instruções de Instalação

Obrigado por descarregares o EuroMillions Master Wizard!

Este programa permite prever números prováveis com base em estatísticas reais do EuroMilhões, utilizando métodos matemáticos como a Cadeia de Markov para aumentar eficazmente as probabilidades de ganhar o jackpot.

🔧 Como instalar:
1. Executa o ficheiro Setup.exe
2. Segue os passos do assistente de instalação
3. O programa será instalado em "C:\Program Files\EuroMillions Master Wizard"
4. Um atalho será criado no Menu Iniciar (e opcionalmente no Ambiente de Trabalho)

É sempre recomendado atualizar o ficheiro "resultados_euromilhoes.xlsx", por cada novo sorteio ocorrido.

🧹 Como desinstalar:
- Vai ao Painel de Controlo > Programas > Desinstalar um programa
- Seleciona "EuroMillions Master Wizard" e clica em "Desinstalar"
OU
- Instala e utiizar o Revo Uninstaller para desinstalar o programa (Recomendado)

📂 Requisitos:
- Windows 10, ou superior
- Não é necessário instalar Python
